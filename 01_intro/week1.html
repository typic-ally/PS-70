This would require me to use a soil moisture sensor, and some kind of programmable device that can control the waterflow of the dispenser. <img src="watering plant design.png" width="604" height="880"> <img src="soil moisture meter.png" width="604" height="880"> <img src="plant meter 2.png" alt="plant meter example 2" width="555" height="555"> As I mentioned before, I am an arts and crafts lover, especially throughout my childhood. One of my favorite things to make as a child was slime, and it was very simple to make. However, this was always a very messy process, so I thought of a fun and useful concept that can benefit children that also enjoy to make slime: a slime machine. Slime is activated when the polyvinyl alcohol in glue and a detergent containing borate ions are combined. This would be a machine that can distribute those substances into a container and handle all of the messy contractions that slime entails: the mixing and distributing process (Glue, borax/detergent, and lotion (to soften the substance). This machine would not be completely hands-free, since there is a kneading process that requires some arm strength and, of course, human hands. Thankfully, the machine should make the overall process less messy, as the stirring and pouring is considered to be the messiest part. The hotdog machine was my inspiration for this concept. The tubes that distributed the condiments on the hotdog caught my interest, and I found it very useful for a mess-free project in the future. <img src="slime machine example.png" alt="bread machine inspo" width="555" height="555"> My last idea is still in process, but I was thinking it should be associated with gaming.
